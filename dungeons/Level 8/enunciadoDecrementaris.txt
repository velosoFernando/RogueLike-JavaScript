LEVEL 8 — DECREMENTARIS, O DEVORADOR DE NÚMEROS

Objetivo:
Criar um programa em JavaScript capaz de decrementar um valor informado pelo usuário.

Regras deste desafio:
1. Crie um input type="number" para informar o valor inicial.
2. Ao clicar no botão, subtraia 1 do valor usando -- ou (valor - 1).
3. Exiba o resultado na tela logo abaixo.
4. Toda a lógica deve estar no arquivo decrementaris.js.
5. O design pode ser modificado, mas um CSS temático acompanha o desafio.
6. O usuário deve escrever todo o código manualmente. Caso utilize LLM, deve reiniciar.

História:
Decrementaris suga energia, reduz poder e desfaz tudo o que cresce.
Ele se alimenta de pequenas perdas, enfraquecendo seus inimigos aos poucos.

Para vencê-lo, domine o ritual da redução contínua.
